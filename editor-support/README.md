# Editor support

Syntax highlighting and filetype detection for Qoz, organised per
editor. Each subdirectory has its own README with install steps.

- [vim/](vim/README.md): Vim 8+ plugin layout (`ftdetect`,
  `syntax`, `ftplugin`).
- [nvim/](nvim/README.md): Neovim notes, including a lazy.nvim spec
  for AstroNvim, LazyVim, kickstart.nvim, and similar configs. The
  underlying syntax file is shared with the Vim plugin.

Other editors (Emacs, VS Code, Helix, Zed) are not covered yet.
Contributions welcome.
